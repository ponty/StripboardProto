==================
RS232 adapter
==================

Status: OK

features:
 - Jumper pins can be used for default Arduino ports.
 - FTDI pinout for RS232 connector

pins
-----

========= =========
FTDI pin  signal     
========= =========
1           gnd     
2           cts
3           5v     
4           rxd    
5           txd    
6           rts     
========= =========
